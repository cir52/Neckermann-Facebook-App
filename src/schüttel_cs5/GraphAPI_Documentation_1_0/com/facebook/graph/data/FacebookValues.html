Classpublic class FacebookValues

public function setImage(value:Object, fileName:String, contentType:String = image/png):void

Allows you to attach an optional image to this request. ote that only one image may be added to a request. Any previous data will be overridden.

Parameters

value:Object — Bitmap, BitmapData, ByteArray or FileReference instance to upload with this request.
 
fileName:String — Name to show on Facebook after this upload is complete.
 
contentType:String (default = image/png) — If you are uploading a ByteArray or FileReference with a custom image format, you can change the contentType to match.
